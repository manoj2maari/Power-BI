# Power BI – Memes Teams Dashboard

This project presents an interactive Power BI dashboard to analyze team-wise employee participation and meme posting activity.

## Project Overview
The dashboard helps understand how employees are distributed across Team Leads (TLs), how many memes are posted, and whether shared links are unique or duplicated.

## Key Features
- Total Employees and Total Memes Posted KPIs
- Team Lead (TL) wise employee distribution
- UNIQUE vs DUPLICATE meme link analysis
- Interactive table with employee details and meme count
- Conditional formatting for link status
- Slicers for Team Lead, Employee Name, and Link Status

## Tools & Technologies
- Power BI Desktop
- Excel
- Power Query
- Basic DAX

## Data Processing
- Cleaned and transformed data using Power Query
- Created calculated measures for counts and summaries
- Applied conditional formatting for better visual clarity

## Outcome
This dashboard demonstrates practical usage of Power BI for data cleaning, interactive reporting, and visual storytelling using real-world structured data.
